提高MAC体验感的软件
发表于 2020-05-23| 更新于 2020-05-25|环境配置
字数总计:2k|阅读时长: 6 分钟
| 阅读量:

因为Mac的python环境乱了，光的py解释路径有9个，根本分不清。剪不断，理还乱，干脆就重装了系统，以后我再也不随便装py了，anaconda这样的虚拟环境也一定要慎用。装完正好把一些不错的应用盘点下，打造一个舒服的环境。本文主题就是白嫖，所有应用均来源于网上，本人不提供软件下载。

浏览器

Safari：原生，没有过多功能，拓展功能也需要掏钱，体验还不好。但耗电量少，出门在外没电时，可以使用。

Chrome：插件丰富，生态好，记忆密码功能方便，但是占用内存较大。

FireFox：插件还行，但记忆密码功能不好用。唯独它的hackbar插件好用，常用它来做ctf题目和抓包。

Microsoft Edge：插件极少，但占用内存小。它是微软抛弃原有edge，基于chromium内核开发的开源浏览器，是微软投入开源市场的尝试。

这次重装配置浏览器时，发现了一个骚操作，Chrome的所有信息都可以导入到Edge里面，包括Edge插件市场中所没有的插件。所以我们可以在Chrome众多的拓展插件中选择好自己所需要的，然后把所有信息导入Edge中，下来就可以转投占有内存小的Edge的怀抱了。你可能会问为什么能这样？我想这可能是因为他们的内核都一样，所以能这样做。

操作方法：

  1. 右上角点击自己登陆后的头像，进入“管理个人资料设置”。

  1. 选择“导入浏览器数据”。

  1. 选择导入浏览器为Chrome，然后勾选所有内容，点击导入即可。

实用工具

Macs Fan Control：控制Mac风扇速度的软件，可以自行设定风扇转速。因为Mac风扇策略很保守，基本都是无声运行，但这样就导致Mac很容易升到一个比较高的温度，而且到了高温时，Mac风扇还是转得不快，温度较高就会影响打字体验。有了它，我们可以根据需要，调整风扇转速。

Tuxera Disk Manager：让Mac能够支持NTFS数据格式的读写。没有它，你可能会发现U盘插到电脑上却无法使用。突然想起来，我的一个U盘是FAT32的，不知道不用这个工具时，能不能实现读写。因为刚开始就装上了这个软件，也没试这种情况。

Archiver：解压缩工具。Mac自带的其实就挺方便的，这个支持的格式更多一点，可作为补充。

Easy New File：Mac的右键拓展，可以像Windows下面那样，方便地在文件夹创建txt，md等文件。

Magnet：分屏工具，可以像Windows下面那样，把窗口拖到边缘是，自动分屏。没有它，你可能得手动规划窗口大小，很不方便。

TotalFinder：Mac下Finder的拓展工具，提供了更多的访达操作。

BarTender3：Mac菜单栏的整理工具，当装完以上的一些工具，你可能发现右上角的菜单栏东西有点多，无法迅速找到所需。它可以把一些不常用的菜单栏图标永久隐藏或者隐藏的另一个菜单栏（解释不清）。等装了，就明白是怎么回事了。

Hands Off!：网络管理软件，能控制其他软件对网络的访问。常用来禁止一个软件的联网，以此来破解软件，防止软件联网检测激活码，下面CodeRunner的激活就要用它辅助。

CleanMyMacX：mac下的磁盘清理软件，还能监控温度，内存占用什么的。刚用mac的时候，还从某宝买了激活码。现在就不怎么用了，因为发现mac没什么必要让这种东西一直后台运行，也不怕病毒什么的，就是清理垃圾可以用一下。可惜了我的几十块钱

迅雷：磁力链下载工具，速度还是可以的，不像百度网盘那么坑爹。

Folx：也是一种磁力链下载工具，有时候会很快，有时候就会用不了，挺迷的。

城通网盘：网盘，许多mac的破解工具都在这个网盘上分享。速度也就一两百KB，撑死。还好那些工具都不大，可以忍受。

Picgo：图床工具，可以实现方便的上传图片到云端，比如github和gitee。写md配图的时候，非常方便。

自动操作：mac自带，会写需求代码，用好了，能省很多事。

文档

Typora：markdown编辑器，可以说是最好用的了。刚开始不熟悉md语法推荐macdown适应一段时间，再用typora。

caj云阅读：用来读取中国知网的caj论文，我么得选择。

XMind ZEN：思维导图软件，Xmind原版和其他的导图软件都试了一下，最后发现ZEN这个版本是最好用的。

PDF Expert：PDF软件，比Adobe的那个更轻量，不过不能复制PDF的文字。要复制文字，可以用Mac自带的预览工具。

Word，Excel，Ppt：微软三件套，不解释。

OneNote：记笔记的东西，可实现手机和电脑同步。

OneDrive：云盘，可以实现和微软三件套的联动操作，就是微软三件套的文档可以自动地同步保存到OneDrve中，防止突发状况没有保存文件。它配合学校发放edu邮箱的1TB空间，简直完美。就是速度可能不太行，这时你需要一个梯子。

Canary Mail：邮箱，界面比较美观，占用内存也还可以。自带Mail操作有点问题，Outlook功能太复杂，Edison那个也不好用。还有一个简洁的邮件客户端是FoxMail，就是张小龙开发的那个，竟然一直在更新。

ScreenFlow：录屏软件，录制的功能较多，现在也没玩过来，只是录制出来的视频体积较大，别的录制软件也不清楚。

Web

Sublime Text：轻量级代码文本编辑工具，也可以配置编译运行代码，适合体量小的代码阅读。

UltraEdit：文本编辑器，但经常用它来看16进制文件，比如说图片。同样的，还有010editer也不错。

Visual Stdio Code(vscode)：轻量级代码编辑工具（相较于vstdio，pycharm等），适合项目开发，全能代码编辑器，不过我至今仍未能配置好它的C++环境。

CodeRunner：代码编辑工具，内置了多种语言的编译器，类似于Windows下的Dev-c和CodeBlocks。单独一个代码文件就可运行，不用创建项目。用它来练习C++和处理PHP语句，是一个不错的选择。

WireShark：一直在更新的抓包软件（抓取监测网卡的所有流量），教程也较多，很不错。

BurpSuite：抓包软件，Web人员必备。

AntSword：Webshell管理软件，类似菜刀，不过界面美观，还有终端shell，执行命令方便。

GitHub Desktop：方便上传文件到git。

CobaltStrike：渗透神器，不过我现在还不太会玩，学习中。

Vmware Fusion：虚拟机软件，用过的都知道。

SQLPro Studio： 数据库连接工具，相较于Navicat来说，体量小，启动快，必要的功能也不少。
